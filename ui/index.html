Storage

Physical disks, SMART health, volumes and space analysis

Drive health forecast

Goes beyond the SMART snapshot — trends the scary numbers (wear, growing read/write errors) over time and scores each drive's risk. Trends build up the more you check. SSD wear needs admin.

Space analyzer

Advanced storage health

Beyond SMART: is TRIM running on the SSDs, any Storage Spaces pool health, the drives' reliability counters, and how much space System Restore's shadow copies are using.

Press Check for TRIM, Storage Spaces, reliability counters and shadow-copy usage.

Network

Adapters, Wi-Fi and diagnostic tools

Bufferbloat test

Measures latency while idle, then while the connection is saturated. A low ping that balloons under load (bufferbloat) is why calls stutter when someone's downloading.
Briefly maxes your connection to measure latency under load.

Domain & website lookup

Registration (WHOIS/RDAP), DNS, hosting, TLS certificate and reputation — to help you judge whether a domain is trustworthy before you trust it.

Only the domain name leaves this PC (to rdap.org, and to VirusTotal if a key is set), plus a TLS handshake to the host. No browsing data is sent.

URL / redirect unmasker

Expand a shortened link and reveal where it really ends up — every hop in the redirect chain — without opening it.

Benchly only requests the URLs — it never runs page scripts. Note: visiting a link can tip off the sender (tracking pixels) or consume a one-time link.

Press Scan to list nearby Wi-Fi networks.

Security

Protection, persistence triage and file reputation

Autostart persistence map signature-checked; suspicious entries first. Click an entry to check it on VirusTotal.

Browser hijack & redirect scan

Checks hosts file, system proxy, and each browser's homepage & default search engine for hijacks.

Remote access & recent changes

After a tech-support scam, an attacker often leaves a remote-access tool and an extra admin account behind. Review anything here you didn't set up yourself.

Trusted root certificate audit

A rogue or interception root certificate lets software silently read your HTTPS traffic. Benchly flags self-signed roots that aren't from a recognised public CA, plus known antivirus / proxy / adware interception roots. “Few certificates” is normal — Windows fetches public roots on demand.

Listening ports

Every port this PC is accepting connections on, the process behind it, and whether that program is digitally signed. Unsigned listeners — especially on a network interface — are worth a second look.

Email header analyzer parsed locally — nothing is sent anywhere

Paste the raw headers of a suspicious email (in most clients: ⋯ → “View source” / “Show original”). Benchly traces the delivery path, the originating IP, the SPF/DKIM/DMARC results, and the classic spoofing tells.

Persistence & exclusions

The hiding spots autoruns flattens or misses — WMI event subscriptions (fileless persistence), services and tasks with suspicious paths, Defender exclusions, and what's executed recently (from Prefetch). Findings are ranked context, not verdicts — legit software trips some of these.

Press Scan to map persistence. Some parts need Run as admin.

Hardening scorecard

High-value Windows hardening checks, each with a reversible one-click fix that documents exactly what it changes. Nothing is applied automatically.

Backup posture

If this drive died today, what would be lost? Benchly audits backups — it doesn't make them.

Pending restart

Windows leaves breadcrumbs when it's waiting on a restart to finish servicing — and until you reboot, new updates and some installers quietly fail. This reads every signal and explains it in plain English.

Press Check to see whether a restart is pending.

Update doctor

Why is Windows Update stuck? This shows the recent update history with the cryptic 0x800f… error codes translated into plain English, plus the health of the services updates depend on. Pairs with the Update-cache reset below.

Press Check to read the update history and decode any failures.

Performance snapshot

“Why is it slow right now?” — samples the machine for 30 seconds and reports the top CPU, memory and disk offenders plus system pressure, ready to copy into a ticket.

Power, sleep & wake doctor

Why won't it sleep? What woke it at 3 AM? What's set to wake it? Plain-English answers from the powercfg dark arts.

Press Check to look at sleep, wake timers and wake devices.

Gremlin hunters

The weird, intermittent stuff: what's hammering the disk when “nothing” is, which USB device keeps dropping, and a button to stamp the moment it just hiccupped.

The instant something hiccups — a freeze, stutter or glitch — hit this. Benchly pulls everything the logs recorded around that moment.
