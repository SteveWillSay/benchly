# Benchly v2.0.1

A small patch: the **self-updater now shows a proper, on-theme progress bar** —
download stage, percentage and megabytes downloaded, with a moving indeterminate
bar when the server doesn't report a size.

(You'll see the new bar on updates *from* this version onward, since it's the running
version that draws the progress.)
