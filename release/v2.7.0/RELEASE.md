# Benchly v2.7.0 — Power, storage & runtime forensics

Bundle H — the last of the v2.4–v2.7 roadmap.

- **Battery & power efficiency** (System) — battery wear and cycle count, plus a short
  `powercfg /energy` trace that names what's hurting power efficiency (admin).
- **Environment & PATH audit** (System) — flags missing-folder, duplicate and quoting
  problems in the Machine/User PATH; one-click clean with the prior value backed up.
- **Installed runtimes** (System) — .NET Framework, .NET (Core/5+), Visual C++
  redistributables and DirectX, for the "missing runtime" diagnosis.
- **Advanced storage health** (Storage) — TRIM status, Storage Spaces, reliability
  counters, shadow-copy (System Restore) space, and the filesystem dirty bit.
- **Audio device doctor** (Devices) — endpoints and their state, audio services, and a
  one-click restart.
