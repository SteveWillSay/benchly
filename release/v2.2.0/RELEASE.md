# Benchly v2.2.0 — the Helper release

A new **Helper** page of friendly, big-button tools for fixing a family member's PC:
text-my-tech-person health summary, "calm this computer down" quiet mode, instant
text-size presets, a camera/mic permission doctor, a BitLocker recovery-key reminder,
a copy-only photo/document rescue, and a paste-and-check "is this a scam?".
