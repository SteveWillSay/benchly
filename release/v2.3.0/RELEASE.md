# Benchly v2.3.0 — Home lab & power user

The last of the v2 roadmap (Bundle D):
- GPU stability & throttle forensics (System) — clocks, throttle reasons, 30-day TDR history.
- Display refresh check (System) — catches a monitor left below its best refresh rate.
- Virtualization health (System) — WSL/Docker virtual disks, Hyper-V switches, VT-x.
- Drive health forecast (Storage) — trends SMART numbers over time and scores drive risk.
- Bufferbloat test (Network) — latency under load, graded.
