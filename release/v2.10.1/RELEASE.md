# Benchly v2.10.1 — Treemap folders tinted by content

The space-analyzer treemap now colours each **folder** by its dominant content type — a folder
mostly full of video is purple, mostly images teal, mostly code green, mostly binaries red, and
so on (the same palette as the File types view). Computed during the existing size walk, so no
extra cost. "What is this drive actually full of?" is now readable at a glance.
