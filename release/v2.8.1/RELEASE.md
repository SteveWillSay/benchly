# Benchly v2.8.1 — Tweaks expansion

Cleanup → Tweaks grew from **25 to 48** reversible, documented toggles. Every one shows its
exact registry location and restores the Windows default when switched off.

- **New Gaming group** — disable mouse acceleration, Game DVR background recording, and
  multimedia network throttling.
- **Performance** — disable transparency effects, window animations, and **Fast Startup**
  (the "did you really do a full shutdown?" fix for dual-boot and driver oddities).
- **Privacy** — disable Activity History/Timeline, Cortana, inking & typing personalization,
  most-used-app tracking, and language-list sharing with sites.
- **Interface** — hide Task View, compact Explorer view, disable Aero Shake, snappier menus,
  full path in the title bar, and **"End task"** on the taskbar right-click.
- **Ads & noise** — hide File Explorer promos, "Finish setting up" nags, the post-update
  welcome page, suggested actions, and Start menu recommendations (also folded into the
  one-click "calm this computer down").
