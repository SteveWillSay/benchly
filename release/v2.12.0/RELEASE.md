# Benchly v2.12.0 — Chevron theme + corporate expansion

## New theme
- **Chevron** appearance (Appearance menu): warm amber-orange on a warm-dark steel palette,
  squared corners, an uppercase wordmark, and a chevron hazard-band across the title bar.

## Corporate (Workplace)
- **Applied policies** tab — a read-only view of every Group Policy / MDM setting actually
  applied to the machine, grouped by area (Windows Update, Defender, Edge, Explorer…) with each
  setting's registry key, type and value.
- **Corporate IT** tab — management & security agents present and their state (SCCM, Intune,
  Defender for Endpoint, third-party EDR, VPN, backup), plus Windows Update management
  (WSUS / Update for Business), proxy, and managed-network config. All read-only.
