# Benchly v1.9.0

**One-click updates.** When a newer version is published, Check for updates now offers
**Download & install** — Benchly fetches the new build, verifies its SHA-256, and restarts
into the new version. Installed copies update in place via the installer; portable copies
swap themselves out and relaunch.

## Downloads
- `Benchly-1.9.0-portable.exe` — single-file portable app
- `Benchly-Setup-1.9.0.exe` — installer (Start-menu shortcut, uninstaller)
- `SHA256SUMS.txt` — checksums (the in-app updater verifies against this)

See `CHANGELOG.md` for the full history.
