# Benchly v1.8.1

**Put any PC on the bench.** — one-stop Windows diagnostics, triage and repair.

A small polish release on top of [v1.8.0](../v1.8.0).

## What's changed

- The glass appearance is now called **Frosted Glass** — same look, a name that actually
  says what it is. Switch to it from the Appearance menu in the title bar, or launch with
  `--theme frost`. (If you'd already saved the old theme, it carries over automatically.)
- The README and the `docs/` set were rewritten in a friendlier, more conversational voice,
  with a gallery of screenshots.
