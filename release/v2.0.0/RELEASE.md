# Benchly v2.0.0

**Put any PC on the bench.** — the "everyday fixes & gremlin hunting" release, first of
the v2 feature bundles.

## What's new

- **Power, sleep & wake doctor** (Toolbox) — why won't it sleep, what woke it at 3 AM,
  and what's armed to wake it, in plain English.
- **Gremlin hunters** (Toolbox) — find what's hammering the disk when "nothing" is, spot
  USB devices that keep dropping, and a "mark the freeze" button that pulls every log
  entry around the moment it hiccupped.
- **Cache & shell repair** (Cleanup → Repair) — one-click fixes for blank icons, broken
  thumbnails, garbled fonts, a dead Store, or a stuck Start search.
- **Printer doctor** (Devices) — catches "offline" printers, a printer that got a new IP
  from DHCP, and duplicate drivers; bring it back online or print a test page.
- **"What changed?"** (Toolbox → baseline) — now spots everyday Windows settings that
  changed (display language, default browser, taskbar, dark mode, mouse, text size) with
  one-click "put it back".
