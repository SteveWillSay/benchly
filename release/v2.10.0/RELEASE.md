# Benchly v2.10.0 — Visual space analyzer

The Storage space analyzer goes visual — WinDirStat-style.

- **Treemap view** — every folder and file is a rectangle sized by the disk it uses and
  colour-coded by type, so the space hogs are obvious at a glance. Click a folder tile to drill
  in; hover for the exact size.
- **File types view** — aggregates every file under the current folder by extension, so you can
  see whether it's video, images, archives or logs eating the space.
- Files are colour-coded by category (images, video, audio, documents, archives, code, binaries)
  using Benchly's existing palette. A **Map / List / File types** toggle switches views; the
  original list (with reveal-in-Explorer and drill-down) is still there.
